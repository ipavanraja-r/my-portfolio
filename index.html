Projects

WiFi Deauther & Packet Monitor

Problem: Unsecured Wi-Fi networks are vulnerable to various attacks, including denial-of-service (DoS) and data interception. There was a need for a practical tool to demonstrate and analyze these vulnerabilities for educational and defensive purposes.

My Role: I designed and developed a custom tool capable of real-time packet monitoring and deauthentication attacks. My focus was on creating a robust and efficient solution for evaluating Wi-Fi network vulnerabilities.

Methodology & Tools: This project utilized Python with libraries like Scapy for packet crafting and analysis. I also integrated the ESP8266 microcontroller for hardware-based deauthentication. The process involved capturing Wi-Fi traffic, identifying connected clients, and sending deauthentication frames to disrupt connections.

Challenges: A key challenge was ensuring the tool operated ethically and legally, focusing on controlled environments. Optimizing packet injection speed and maintaining stability across different Wi-Fi adapters were also important.

Results/Impact: The tool successfully demonstrated the ease with which deauthentication attacks can be performed, highlighting critical Wi-Fi security flaws. It served as an effective educational resource for understanding 802.11 protocol weaknesses and the importance of strong network security configurations. This project significantly improved my practical understanding of wireless security.

Secure Web Application Framework

Problem: Many web applications are susceptible to common vulnerabilities like SQL injection and XSS due to insecure coding practices and lack of robust security features in development.

My Role: I led the development of a secure web application framework using Flask, focusing on integrating strong security measures from the ground up. My responsibilities included designing secure authentication flows and database interactions.

Methodology & Tools: This project leveraged Python with Flask for the backend, focusing on ORM for SQL injection prevention, content security policies (CSP) for XSS mitigation, and secure session management. We implemented advanced authentication mechanisms including multi-factor authentication (MFA) to bolster user security.

Challenges: Balancing development speed with rigorous security implementation was a challenge. Ensuring all third-party dependencies were vetted for vulnerabilities and maintaining consistent secure coding standards across the team required meticulous effort.

Results/Impact: The framework significantly reduced the attack surface for common web vulnerabilities, making it a reliable foundation for building secure web applications. It served as a proof-of-concept for integrating security into every stage of the development lifecycle, leading to a 30% reduction in identified critical vulnerabilities during internal security audits.

P4wnP1 Pwnagotchi Enhancements

Problem: Standard P4wnP1 and Pwnagotchi implementations, while powerful, often lack specific customizations needed for advanced WPA/WPA2 network exploitation and efficient password cracking.

My Role: I customized and enhanced the firmware for P4wnP1 and Pwnagotchi devices. My objective was to improve their capabilities for WPA/WPA2 network exploitation, specifically focusing on increasing password cracking efficiency.

Methodology & Tools: This project involved deep dives into the underlying Linux operating system on the Raspberry Pi (used by Pwnagotchi) and the firmware of the P4wnP1. I utilized custom scripts (Bash and Python) to modify system behaviors, integrate advanced handshakes capturing tools (like Bettercap), and optimize the data collection for offline cracking with tools like Hashcat.

Challenges: A significant challenge was ensuring stability and compatibility across different versions of the firmware and hardware. Debugging network stack interactions and optimizing resource usage on embedded devices required extensive trial and error.

Results/Impact: The custom firmware led to a measurable improvement in the success rate of WPA/WPA2 handshake captures and significantly boosted the efficiency of subsequent password cracking attempts by streamlining data transfer and processing. This project expanded my expertise in embedded systems security and wireless attack vectors.
